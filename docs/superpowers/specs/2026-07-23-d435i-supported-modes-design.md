# D435i 지원 스트림 모드 README 문서화 설계

## 목표

연결된 Intel RealSense D435i에서 확인한 지원 스트림 프로파일을 README에 명시하여, 지원하지 않는 해상도·FPS 조합으로 인한 `Couldn't resolve requests` 오류를 예방한다.

## 범위

- RGB, Depth, 좌·우 IR 스트림을 모두 문서화한다.
- 캡처 코드에서 요청하는 형식만 다룬다: RGB `bgr8`, Depth `z16`, IR `y8`.
- 해상도별 FPS는 하나의 셀에 묶어 중복을 줄인다.

## README 구성

`3DGS용 RGB 영상 캡처` 섹션에서 기본 설정 설명 다음에 `D435i 지원 스트림 모드` 소제목을 둔다. RGB, Depth, IR 표를 분리한다. IR 1과 IR 2는 동일한 모드를 제공하므로 하나의 표로 정리한다. `512×512`는 지원하지 않으며, 정사각형 출력이 필요하면 지원 모드로 촬영한 다음 크롭 또는 리사이즈하도록 안내한다.

## 검증 기준

- 표의 모든 항목은 2026-07-23에 `pyrealsense2`로 조회한 D435i 프로파일과 일치한다.
- RGB 표에는 60 FPS에서 사용 가능한 해상도와 30 FPS 전용 고해상도 모드를 구별해 표시한다.
- README의 기본 캡처값 1280×720, 30 FPS가 표에 포함된다.
